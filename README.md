# Retail Sales Performance Dashboard
This project is a Power BI dashboard designed to analyze retail performance and support decision-making.
## Overview
The dashboard tracks key retail metrics and explains performance trends using automated storytelling.

It helps answer:
- What is happening?
- Why it is happening?
- What should be done next?
## Key Features
- KPI tracking:
  - Total Sales
  - Achievement %
  - YoY %
  - Conversion %
  - Entry Rate %
  - Traffic YoY %

- Weekly performance analysis
- Comparison with target and last year
- Automated insights using DAX
- Interactive filtering by week
## Key Insight Example

Sales decreased week-over-week and remain below target.  
However, conversion improved significantly, indicating stronger in-store effectiveness and potential recovery.
## Business Value
In many retail environments, data is collected manually using spreadsheets, which is time-consuming and inefficient.
This dashboard replaces manual reporting with:
- Automated calculations
- Clear insights
- Faster decision-making
## Tech Stack
- Power BI
- DAX
- Python (for data generation)
## Dataset
The dataset is synthetically generated using Python to simulate a real retail environment.
It includes:
- Weekly sales
- Targets
- Foot traffic (inside and outside)
- Conversion metrics
